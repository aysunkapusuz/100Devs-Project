## How it's made:
<strong> Tech used: </strong> Node.js, Express.js, Handlebars, MongoDB & Google OAuth 

I code along with Traversy Media's Tutorial in order to learn Google OAuth strategy and more.

## Usage

Add your mongoDB URI and Google OAuth credentials to the config.env file

```
# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start
```
